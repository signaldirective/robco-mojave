# RobCo Omarchy Theme

An Omarchy Theme for your Arch Linux / Hyprland setup based on the RobCo
Terminals found in Fallout: New Vegas.

![RobCo Omarchy Theme Screenshot](./preview.png)

# Installation

To install this theme, simply use the ``omarchy-theme-install``   

command: `` omarchy-theme-install https://github.com/signaldirective/robco-mojave``
